# clara-ai-pipeline

AI pipeline for processing and managing transcripts, accounts, and workflows.

## Project Structure

```
data/transcripts/       # Raw transcript files
  demo/                 # Demo transcripts
  onboarding/           # Onboarding transcripts
outputs/accounts/       # Processed account outputs
scripts/                # Automation and utility scripts
workflows/              # Workflow definitions
changelog/              # Change history
```

## Setup

1. Copy `.env.example` to `.env` and fill in your values
2. Install dependencies as needed per script requirements

## Usage

See individual scripts in `scripts/` for usage instructions.
